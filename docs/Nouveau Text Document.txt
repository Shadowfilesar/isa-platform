## `00-README_FIRST.md`

أي شخص يفتح المشروع يقرأ هذا الملف أولًا قبل أي شيء.

سيكون فيه:

```text
STOP.

Do NOT write code.

Read these documents first:

1. ISA_PROJECT_RULES.md
2. ISA_ARCHITECTURE.md
3. FILE_OWNERSHIP.md
4. DEPENDENCY_MAP.md
5. AI_DEVELOPMENT_PROTOCOL.md

Only then start development.
```

## بعد هذه المجموعة

سيصبح مشروع ISA ليس مجرد مشروع Laravel، بل مشروع موثق بطريقة احترافية جدًا. أي مطور جديد أو أي نموذج ذكاء اصطناعي سيتمكن من فهم القواعد، المعمارية، وآلية العمل قبل أن يكتب أول سطر كود، وهذا سيقلل الأخطاء بشكل كبير ويجعل التطوير أسرع وأكثر استقرارًا.
